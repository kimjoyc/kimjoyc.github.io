Tinker9 Project

The experimental data showcases the scaling and performance analysis of a simulation using different numbers of replica exchanges (1, 2, 4, and 8), revealing their impact on the root mean square deviation (RMSD) values, which quantify the structural deviation from a reference structure. The results demonstrate that increasing the number of replica exchanges leads to reduced RMSD values, indicating improved agreement with the reference structure and higher accuracy in reproducing desired structural characteristics. This suggests that employing a greater number of replica exchanges can enhance the fidelity and accuracy of the simulation by effectively exploring the conformational space.

ANI-1 Model

The investigation focuses on reproducing the ANI1 Neural Network Potential (NNP) for molecular systems using the NNP-ANI model, which consists of subnetworks corresponding to each atom (H, C, N, O). Hyperparameter tuning is required, involving adjustments to the learning rate, batch size, epochs, and early stop. Features for the model are engineered through the calculation of atomic environment vectors (AEVs), which are highly-modified versions of Behler and Parrinello symmetry functions. Cross validation is performed by summing errors between model predictions and actual energy using a custom training class. The model is evaluated using mean squared error (MSE) and RMSE metrics, and overfitting is assessed by comparing training and validation errors. To improve the model, options include adjusting the network depth, changing activation functions, or exploring alternative approaches. For inference, sufficient analysis and visualization are conducted, including hypothesis testing with a defined test statistic, significance level, and justification for the chosen procedure. Additionally, the performance of the model is checked using errors, and statistical tests are selected based on the specific case, considering the limitations of using randomly selected coordinates for AEVs instead of representative molecules.
